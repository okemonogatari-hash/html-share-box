よくある質問（はじめての人向け）

「そもそも何なの？」から「お金かかるの？」まで。
みんなが気になることを、なるべくやさしい言葉で答えます。

Q そもそも Claude Code って何？普通のClaudeと何が違うの？
ひとことで言うと、「おしゃべり相手」と「作業してくれる人」の違いです。

普通のClaude（ブラウザで使うチャット）は、質問に答えてくれる「相談相手」。
Claude Code は、あなたのパソコンの中に入って実際にファイルを触って作業してくれる「助手さん」です。

💬 普通のClaude（チャット）

「この書類どう直したらいい？」
→ アドバイスをくれる
→ 直すのは自分

🧠 Claude Code

「この書類直しといて」
→ ファイルを開いて
→ 直接直してくれる

Q ChatGPTとかで十分じゃないの？わざわざ別のもの使う必要ある？
「聞くだけ」で済む用事なら、ChatGPTで十分です。

でもたとえば、こんなことを想像してみてください。

💬 ChatGPTの場合

あなた：「ファイル名を統一したい」
AI：「こうやって変更してください」
あなた：100個のファイルを手動でひとつずつリネーム

🧠 Claude Codeの場合

あなた：「このフォルダのファイル名、統一して」
AI：100個全部、一瞬で変更完了
あなた：何もしない

つまり
ChatGPTは「教えてくれる先生」、Claude Codeは「実際に手を動かしてくれるアシスタント」。
用途が違うので、両方使い分けるのがいちばん賢いです。
Q プログラミングできないけど、本当に使えるの？
使えます。むしろ「できない人」にこそ価値があります。

Claude Codeに必要なのは「日本語で、何をしてほしいか伝える力」だけ。
「このフォルダ整理して」「この表をきれいにして」「会議の録音を文字にして」
――こういう普通の日本語で、ぜんぶ動きます。
実際の例
おけもんはプログラミング未経験ですが、Claude Codeと一緒に
Webサイト公開、自動文字起こしシステム構築、150人イベントの企画まで実現しています。
Q お金はいくらかかるの？無料では使えない？
Claude Code を使うには、月額 $20（約3,000円）のProプランが必要です。

無料のClaudeチャット（ブラウザ版）で「こういうことできるんだ」と試してみてから、
「もっとやってもらいたい！」と思ったらProに上げるのがおすすめ。
料金のイメージ
月3,000円で「24時間働いてくれる秘書」が手に入ると考えたら、
かなりおトクだと思いませんか？
もっとヘビーに使いたい人には $100/月のMaxプランもあります。
Q 自分のファイルをAIに見せて、情報が漏れたりしない？
基本的に安全です。あなたのデータがAIの学習に使われることはありません。

しかも、Claude Codeはファイルを読むときに毎回あなたに許可を求めます。
勝手にファイルを開いたり、送信したりすることはできません。

家の鍵を渡すのではなく、「この部屋だけ見ていいよ」と指定できる感じです。
Q スマホでも使える？
Claude Code自体はパソコン専用です。

スマホでは普通のClaude（チャット版）が使えます。
「会話で相談する」はスマホ、「作業してもらう」はパソコン、と使い分けるのがおすすめ。
Q WindowsでもMacでも使える？
はい、両方OK。Macが一番かんたん（1分で準備完了）。
Windowsでも使えますが、最初にちょっとだけ設定が必要です。
Q 何を用意すればいい？始めるのに必要なものは？
たった3つだけ。

1. パソコン（Mac、Windows、どちらでもOK）
2. インターネット環境
3. Claudeのアカウント（メールアドレスで無料作成）

ハイスペックなパソコンは不要。普段使いのノートPCで十分です。
Q Cursor とか Copilot とか他にもAIツールあるけど、違いは？
ざっくり言うと、「誰向けか」が違います。

Cursor・Copilot → プログラマー向け。コードを書く人が速く書くための道具。
Claude Code → プログラマーじゃなくてもOK。日本語で話しかけるだけで作業してくれる。

「コードを書ける人がもっと速く書く」のか、「コードを書けない人でも使える」のか。
この違いが一番大きいです。
Q どういう人に向いてるの？
「パソコンで繰り返し作業をしている人」は、ほぼ全員向いてます。

特に嬉しいのはこんな人：
• ファイルの整理やデータ入力に時間を取られている人
• 「こういうツール欲しいけど、作れない…」と思ったことがある人
• ひとりで仕事を回していて、秘書が欲しい個人事業主
