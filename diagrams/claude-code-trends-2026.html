3月24日、大きめのアプデが3つ来た

🙌 NEW

毎回「許可しますか？」を
押さなくてよくなった

今まではAIが何かするたびに「いいですか？」と聞かれていたけど、安全な操作は自動でOKしてくれるように。危ないものだけ確認が来る。作業がサクサクに。

📱 NEW

スマホから
AIに指示が出せるように

外出中でも、スマホのアプリからClaude Codeに指示が出せる。「帰ったらパソコンの前で…」が不要に。思いついた時にすぐ頼める。

🖥️ NEW

AIがパソコンの画面を直接操作できるように

Canvaやスプレッドシートなど、「AIが直接触れなかったツール」もマウスとキーボードで操作してくれる。API？ 知らなくて大丈夫。

✦

他にもこんなことが

🗣️ 音声

声だけでAIに指示できる

キーボードを打たなくても、話しかけるだけでOK。日本語対応。手が塞がってる時や、考えながら伝えたい時に便利。

⚡ 活用例

Excel集計4時間 → 20分に

コードを書かなくても、CSVを渡して「集計して」と言うだけ。レポート作成・データ分析が非エンジニアでも一瞬に。

🔰 触ってみたいなと思ったら

パソコンの「ターミナル」を開いて、1行コピペするだけで始まります。

1. Claude Pro に登録 2. ターミナルを開く 3. インストールコマンドを貼る
