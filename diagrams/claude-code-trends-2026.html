BREAKING

Auto Mode

AIが操作の安全性を自動判定。ルーチン操作は自動承認、危険な操作だけ人間に確認。

毎回 yes を押す地獄から解放。

BREAKING

Computer Use
（Cowork）

CoworkがMacのキーボード・マウスを直接操作。API非対応のアプリもスクリーン操作でカバー。

スマホ（Dispatch）からも指示可能。

Channels

Discord・TelegramからClaude Codeを操作。

PCの前にいなくても開発指示が出せる時代に。

動画自動生成

Remotion x Agent Skills

プロンプトだけで動画を自動生成。29のベストプラクティスをスキルパック化。ブログ → 紹介動画を約50秒で生成。字幕付き動画もOK。コンテンツ発信が劇的に変わる。

非エンジニア活用

「全員が使うべきツール」の声

5大活用シーン: CSVデータ分析・競合調査・議事録処理・週次レポート・メール返信。Excel集計 4時間 → 20分（コード記述なし）。Google Ads分析100項目チェック自動化の実例も。

音声入力

Voice Mode

/voice コマンドでPush-to-Talk音声入力を起動。日本語含む20言語対応。手が塞がっている時の指示出しや、複雑な要件のディクテーションに。

エコシステム爆発

Skills / Plugins マーケットプレイス

GitHub上に340+ plugins、1367+ skillsが公開。マーケットプレイス（claudemarketplaces.com）が登場。Anthropic公式17スキルもオープンソース公開。

ワークフロー

Claude Code x n8n ワークフロー構築

非エンジニアがClaude Codeでn8nの自動化ワークフローを構築する事例が登場。複数SaaSを束ねるノーコード自動化がClaude Code一本で実現可能に。
