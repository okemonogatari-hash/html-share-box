デスクトップ版 Claude Code

はじめてのセットアップ
完全ガイド

「Gitがない」「Homebrewがない」の警告も全部解決します
ターミナル版ではなく、デスクトップアプリ版のためのガイドです

📋 始める前に確認しておくこと
💻
Macの場合
macOS 13（Ventura）以降が必要です。「りんごマーク → このMacについて」で確認できます
👤
Claudeのプランが必要
無料プランでは使えません。Pro または Max プランが必要です（月額 $20〜）
📶
ネット環境が必要
インストール中も使用中も、インターネット接続が必要です
⏱️
準備時間の目安
ダウンロードやインストールを含めて、だいたい20〜30分みておきましょう
セットアップの流れ（全7ステップ）
1
STEP 1 ・ ダウンロード
公式サイトからアプリをダウンロードする

ブラウザで claude.ai を開いて、ログインした状態でアプリをダウンロードします。

💡 ダウンロードしたファイル（.dmg）は「ダウンロード」フォルダに入ります。ダブルクリックして開いてください。
↓
2
STEP 2 ・ インストール
アプリをApplicationsフォルダに移動する

開いたウィンドウの中で、「Claude」アイコンを右の「Applications」フォルダにドラッグ＆ドロップします。

移動が終わったら、Finderの「アプリケーション」フォルダから「Claude」をダブルクリックして起動します。

⚠️ 「"Claude.app"は開発元を確認できないアプリです」と表示されたら →「開く」をクリックすれば大丈夫です（Apple公証済みの安全なアプリです）
↓
⚠️ ここで警告が出ることがある！あわてないで
よくある警告 ①
「Gitが見つかりません」「ローカルセッションにはGitが必要です」
Gitというファイル管理の道具がMacに入っていない状態です。次のSTEP 3で解決します。
よくある警告 ②
「Homebrewが見つかりません」
Homebrewというアプリのインストールを助けてくれる道具が入っていない状態です。STEP 3 → 4の順に解決します。
よくある警告 ③
「コマンドラインデベロッパツールをインストールしますか？」（Macのポップアップ）
「インストール」を押すと裏でダウンロードが始まります。8〜10分かかるので、そのまま待ちましょう。
↓
3
STEP 3 ・ Homebrew インストール
「道具を入れる道具」Homebrew を入れる

Homebrew（ホームブリュー）は、Macに色々な道具を簡単に入れてくれる「インストーラー」のようなものです。まずこれを入れます。

Macの虫眼鏡アイコン（🔍）→「ターミナル」→ Enterで黒い画面を開いて、以下をコピー＆ペーストしてEnterを押します：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
⚠️ Macのパスワードを聞かれます。普段ログインに使うパスワードを入力してください（入力しても画面に文字が表示されませんが、ちゃんと入力されています）
💡 M1・M2・M3チップのMac（2020年以降の薄いMac）の場合、インストール後に追加で以下の2行も実行してください：

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc

source ~/.zshrc
↓
4
STEP 4 ・ Git インストール
Claude Codeに必要な「Git」を入れる

Git（ジット）は、ファイルの変更履歴を管理する道具です。Claude Codeが作業するために必要です。Homebrewが入れば、簡単に追加できます。

同じくターミナルで以下を入力してEnterを押します：

brew install git

完了したら確認してみましょう：

git --version
# → 「git version 2.○○.○」と表示されればOK！
✅ バージョン番号が表示されたら、Gitのインストール完了です！
↓
5
STEP 5 ・ ログイン
Claudeアカウントでログインする

アプリを（再）起動すると、ブラウザが自動で開きます。Claude.aiのアカウントでログインして「許可する」をクリックすると、デスクトップアプリに自動で戻ります。

💡 ブラウザが開かない場合は、アプリの画面で「C」キーを押すとURLがコピーされます。それをブラウザのアドレスバーに貼り付けてアクセスしてください。
❌ 「Pro/Maxプランが必要です」と表示された場合は、claude.ai からプランをアップグレードしてください。無料プランでは使えません。
↓
6
STEP 6 ・ Code タブを開く
アプリ上部の「Code」タブをクリックする

ログインが完了すると、アプリ上部に「Code」というタブが表示されます。ここをクリックすると、Claude Code の画面が開きます。

💡 Claudeの通常チャット画面とは別の場所です。「Code」タブが見当たらない場合は、アプリを再起動してみてください。
↓
7
STEP 7 ・ フォルダ選択
作業したいフォルダを選んでスタート！

「フォルダを選択」（Select folder）ボタンをクリックして、Claudeに整理・作業してほしいフォルダを選びます。

最初は「デスクトップ」や「ダウンロード」フォルダなど、使い慣れた場所から試してみるのがおすすめです。

✅ フォルダを選ぶと会話欄が出てきます。日本語でそのまま指示を入力するだけでOKです！
↓
🎉

セットアップ完了！

あとはやってほしいことを日本語で話しかけるだけ。
「このフォルダのファイルを整理して」「〇〇のリストを作って」など、なんでも頼んでみてください！
