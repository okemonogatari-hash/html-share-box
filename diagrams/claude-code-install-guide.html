Claude Code はじめの一歩

「インストールしたのに
反応がない！」→ 大丈夫！

赤いエラーとポップアップ、どう対処すればいい？
ターミナル初心者でもできる解決方法をまとめました

まず知ってほしいこと

なぜ赤いエラーが出たの？

🧰
Claude Codeには「道具箱」が必要
「git」という、ファイルを管理するための道具が必要です。Macに最初から入っていない場合があります。
⏳
「インストール」を押したら、裏で準備中！
ボタンを押したら反応がないように見えても、画面の裏でちゃんとダウンロードが始まっています。8〜10分かかります。
✅ 「インストール」を押していたなら、今も準備中の可能性が高いです！
焦らず、まず10分待ってみましょう
確認の手順（3ステップ）
1

10分くらい待ってみる

「インストール」を押していたなら、裏でダウンロードが進んでいます。Wi-Fiの速度によっては少し時間がかかります。待っている間はMacを閉じないようにしましょう。

💡 インターネットの速度次第で、3分〜15分かかる場合があります
↓
2

ちゃんと入ったか確認する

Mac右上の虫眼鏡アイコン（🔍）をクリックして「ターミナル」と入力 → Enterキーで開きます。
黒い画面が開いたら、以下をそのままコピー＆ペーストしてEnterキーを押してください：

git --version
✅ 「git version 2.○○.○」のような文字が出たら完了！
❌ 何も出ない or エラーが出たら → 次のステップへ
↓
3

Claude Codeを再起動する

確認できたら、Claude Codeを一度閉じて、もう一度開いてみましょう。これで赤いエラーが消えるはずです。

🎉 これで完了です！お疲れさまでした

⚠️ まだダメなら → 手動でインストールしてみよう

1
Mac右上の虫眼鏡アイコン（🔍）→「ターミナル」と入力→ Enterで開く
2
黒い画面に xcode-select --install と入力してEnterを押す
（コピー&ペーストでOK。半角スペースとハイフン2つに注意）
3
ポップアップが出たら「インストール」→「同意」をクリック
4
10分ほど待つ（Wi-Fiを切らずに待つ）
5
ターミナルで git --version → バージョンが出たら完了。Claude Codeを再起動！
