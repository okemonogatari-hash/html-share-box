Claude Code 活用法

AIを「社員」にしたら、
会社が24時間動き続けた

Claude Codeのフォルダー分けとルール設定で、5人の役員チームを作る方法。
1人で複数の仕事を同時に進める仕組みを図解します。

2
3種類のルールで、AIの動きを育てる
「毎回同じことを教えなくていい」仕組みをルールとして書き込む
🏢
CLAUDE.md — 全社共通ルール
Claude Codeを起動するたびに自動で読み込まれる「会社憲法」。報告ルール・情報管理・品質基準など、全員に守ってほしいことを書く。3,000文字以内に絞るのがコツ。多すぎると言うことを聞かなくなる。
例：「成果物は必ず指定フォルダに保存する」「事実と意見を分けて書く」
📁
フォルダー別ルール — 部署専用の指示
CMOフォルダにだけ、CFOフォルダにだけ——部署ごとの細かいルールはここに。全社ルールより長文でOK。対象が絞られているので、迷わず動いてくれる。
例：CMOなら「集客導線を意識した文体で書く」「競合調査は必ず参考資料フォルダに保存」
⚡
スキル — 呼び出せる専門家の知識
「YouTube台本の専門家」「Excel処理の専門家」など、特定の仕事の得意技をスキルとして登録。作業を依頼すると自動でそのスキルが呼び出される。毎回プロンプトを書き直さなくていい。
例：YouTube台本スキル → 「伸びやすい構成」「話し方のトーン」「尺の目安」が自動適用
