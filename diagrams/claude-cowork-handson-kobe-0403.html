— あなたが使うのはどっち？

この違いを理解するだけで、何ができるかのイメージがガラッと変わる。

チャット型AI

☕ スタバの店員

例：ChatGPT、Claudeのチャット画面

🛠️ ツール紹介

CoworkとClaude Code
— 実は同じことができる、別々の入口

どちらが上とか下とかじゃない。入口の形が違うだけ。どちらから始めても同じフォルダにアクセスできる。

🌸

Cowork

画面操作タイプ
初心者フレンドリー ポップアップで選択肢 月1.5〜1.6万円
「何しますか？」って親切に聞いてくれる。ポチポチ選ぶだけで進められる。
ただ、その親切さの分、トークン（スタミナ）の消費が多め。上限に達しやすい。
⌨️

Claude Code

文字指示タイプ（ターミナル）
テキストで指示 トークン効率◎ $100〜$200/月
黒い画面（ターミナル）でテキストを打って指示する。慣れると圧倒的に速い。
最初は戸惑うかもだけど、慣れれば何でもできる。おけもんはこっちメイン。
💡 共通してできること： どちらもフォルダを指定すれば同じファイルを見に行ける。おけもんはCoworkとClaude Codeを同じフォルダに向けて使ってるので、どちらから操作しても結果が統合される。
✋ ハンズオン実況

当日、みんなで実際にやったこと

「インストール」から「ブラウザ操作」まで、手を動かしながら進んだ。神戸は通信が快適でスムーズに完了！

STEP 1 📥 インストール＆アカウント作成
Claudeアプリをダウンロード → Macの場合はApplicationsフォルダへドラッグ。Googleアカウントで登録OK。電話番号も設定。
神戸のポイント：前日の梅田オフィスではダウンロードだけで30分以上かかった人も。今日の神戸オフィスは通信が快適で全員スムーズに完了！
STEP 2 📁 フォルダを「AIのお家」に指定
デスクトップに「日付＋練習用」フォルダを作成。これをCowork / Claude Codeに指定することで、AIがそのフォルダを見に行けるようになる。
確認できたこと：「旅行計画を作って」「3月の経費をExcelにまとめて」と言うと、実際にFinderで確認できるファイルができた！
STEP 3 🌐 Chrome連携デモ
Claude CodeにChrome拡張機能を入れるとブラウザを操作できるようになる。オフ会チャットに貼った指示文をコピペするだけで設定完了。
デモ：「明日（4/4）の10時〜12時で神戸オフィスを予約して」と入力 → AIが実際に予約ページを探して操作しにいった！（うまくいくケース・いかないケースあり）
STEP 4 🔗 MCPコネクタで他ツールと繋ぐ
Coworkの「カスタマイズ」→「コネクタ」から各種サービスと連携。Google Drive、Googleカレンダー、Notion、Slack、GitHubなど多数が公式対応。
参加者から出た話題：WordPress、Note、Chatworkの対応状況を確認。WordPressは連携◎、NoteはAIに確認要、Chatworkはリストにないので別途調査。

❓ Q&A

当日出た質問、全部答える

「これ私も気になってた！」という声がいっぱい出た。

Q
ChatworkはMCPコネクタのリストに見当たらないのですが…
A
CoworkのコネクタリストにはChatworkの公式対応はない。Claude Codeのチャット画面で「Chatwork MCP使えますか？」と直接確認するのが一番早い。非公式で使えるケースもある。
Q
Noteってコネクタで連携できるんですか？
A
公式MCPの対応はまだないが、非公式APIで連携できる可能性がある。Claude Codeに「Noteと連携したいんだけど」と聞いてみると最新情報を教えてくれるので、試してみて。
Q
トークンって何？上限がきたらどうなるの？
A
トークンは会話するたびに消費される「スタミナ」みたいなもの。上限は5時間ごとにリセットされる（例：11時に上限→16時にリセット）。「5時間枠」と「週間枠」の2段階ある。設定の「使用料」で現在の消費量が確認できる。
Q
WordPressで自動投稿はできますか？
A
できる。MCPでWordPressと連携すれば、記事の自動投稿・リライトが可能。ライターさんやブロガーさんには特に使えるはず。
Q
他人のカレンダーや他人のアカウントも見られてしまうの？
A
連携できるのは自分がログインしているアカウントのみ。他人のカレンダーや他人のGmailは見られない。セキュリティ上も自分のアカウント範囲内でしか動かない。
Q
一度成功した指示をまた使いたい時はどうすればいい？
A
成功した指示のやり方は必ずファイルに記録しておくこと！翌日同じように試してもうまくいかないケースがある。「この作業の手順」として自分のフォルダに保存しておけば、次回AIに読み込ませて同じように動かせる。
Q
音声入力って精度どうなの？
A
Claude純正のマイク入力は精度が高く、ざっくりした話し言葉でも意図を汲み取ってくれる。ただしClaude画面上でしか使えない。おけもんはAquaVoice（月1,500〜1,800円）を使ってて、全アプリで音声入力できる。早めに慣れる価値あり！
