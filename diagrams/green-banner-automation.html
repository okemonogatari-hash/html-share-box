今の作業と、自動化後の世界。

AS-IS

今の進め方

  • Green求人URLを1本ずつブラウザで開く
  • 仕事内容・年収・企業情報を手でコピペ
  • 職種ごとにスプシに手貼りしてまとめる

  • Green求人URLのリストを渡すだけ
  • AIが各ページを巡回して原稿を自動抽出
  • 職種ごとに自動で整理・グルーピング
  • バナー用コピー3案 + テイスト指示まで生成
  • 決まったフォーマットのスプシに自動で書き出し
感覚値：URLを投げて、数分〜翌朝にはスプシが埋まっている。
02 / How it works

裏側で何が起きているか。

魔法ではなくて、それぞれの工程に向いたAI・ツールを組み合わせているだけです。
手作業だと重いところを、機械が得意な形に分解して流してます。

STEP01

求人ページを自動で開いて、中身を読み取る

渡していただいたGreen求人URLを、ブラウザ自動操作ツールが順番に巡回。ページの中身をまるごと取得します。手でページを開く作業がゼロに。

使用技術：Playwright（ブラウザ自動操作）
STEP02

求人原稿を「使える形」に整理する

職種名・仕事内容・年収レンジ・勤務地・求人の"売り"・想定ペルソナを、AIが構造化して抜き出します。ここがいちばん時間を溶かしていた下ごしらえ工程。

使用技術：Claude + Groq（大量テキスト要約）
STEP03

職種ごとにバナーコピー案を生成する

職種のターゲット層・求人の訴求ポイントを踏まえて、タイトル案・キャッチ案・テイスト指示・挿入テキストを3パターン自動生成。人のレビューは残しつつ、ゼロから書く手間は消えます。

使用技術：Claude（日本語品質重視）
STEP04

いつものスプシに、自動で書き出す

既にお使いのスプシ（週次シート・職種別構成）のフォーマットをそのまま踏襲。週次シートの自動採番、職種ブロックの自動生成までやります。デザイナーさんへの流し方は今までと同じでOK。

使用技術：Google Sheets API
