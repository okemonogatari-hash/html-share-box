03 / アンカー

AIが逆立ちしても作れない「アンカー」を持て

「アンカー」は、AIがマネできない本物の部分のことです。 これからはAIをどれだけ使ってもいい。でも最後に、コピーできない本物を入れないと弱くなります。

ANCHOR 01

土地

リアルな場所のこと。たとえば学校の体育館、地元の商店街、その会場。AIは画像は作れても、その場所自体は作れません。

ANCHOR 02

時間

長く積み上げた時間のこと。たとえば3年間の部活、ずっと続けた活動、先に取った場所。あとから一気に作れません。

企画の中心に置く

非代替の核

AIで作るスピードを上げるのはOK。
でも最後は、「その学校だから」「その人たちだから」できる要素を入れる。

企画ごとに最低1つは入れると強い。

ANCHOR 03

思い出

その場にいた人だけが持つ記憶。たとえば修学旅行、最後の大会、卒業前の時間。これはコピペできません。

ANCHOR 04

制作過程・関係性

何を作ったかだけでなく、みんなで準備した時間、裏側、協力した流れそのもの。完成品だけより、そこが刺さることも多いです。

ANCHOR 05

ブランド

「この人に頼みたい」と思われる力。たとえばあの先輩なら安心、あの先生なら任せたいという信頼です。

図解で言い切るなら: AI時代の勝ち方は、作るところはAIで速くして、最後の決め手は「本物の思い出」や「その場らしさ」で作ることです。

05 / 必須能力

これからのクリエイターに必要な6つのコアスキル

ここも難しく考えなくて大丈夫です。要するに、AIに任せられない「人っぽい力」を伸ばそう、という話です。

CORE 01

審美眼
価値認識力

たくさんの案の中から、「これが一番いい」と見抜く力。文化祭のポスターなら、目立つだけでなく人が見たくなる案を選べる力です。

例: 青い屋台は目立つけど食欲はわきにくい、と分かる感覚
CORE 02

目的設計力

「そもそも何のためにやるの?」を決める力。ポスターを作る前に、「集客したいのか、かわいく見せたいのか」を決める力です。

例: ゴールを決めるのは人、作業を速くするのがAI
CORE 03

曖昧さ耐性
ドライブ力

まだ答えがなくても「じゃあこれでやってみよう」と決める力。部活でも、ずっと相談するだけより、まず動く人が強いです。

例: 完璧を待つより、仮で進めて後で直す
CORE 04

人間力
明るさ

一緒にいると空気がよくなる力。あいさつできる、感じがいい、話しかけやすい。これは想像以上に大事です。

例: 同じ実力なら「また一緒にやりたい人」が勝つ
CORE 05

ストーリー構築力

「この企画にはこんな意味がある」と語れる力。卒業前のイベントなら、ただ楽しいだけでなく「最後の思い出」にできる力です。

例: 物ではなく、心が動く理由を作る
CORE 06

非合理を守る力

効率だけでは説明できない大事なものを守る力。たとえば、みんなで準備した時間、雑談、打ち上げ、裏話みたいな部分です。

例: 最短ではなくても、思い出が残るやり方を選べる
06 / 実践

明日からの打ち手 5ステップ

じゃあ何をすればいいのか。中学生でも始めやすい形にすると、この5つです。

01

AIの得意を体感する

まずはChatGPTや画像生成を触ってみる。作文、要約、ポスター案づくりで使ってみると、「AIはここが速い」が見えてきます。

知らないことが一番危ない
02

自分のアンカーを書き出す

自分しか持っていないものを書き出す。部活経験、地元、友達との思い出、得意分野、関係性。そこがあなたの強みになります。

差は能力より所有から生まれることも多い
03

成功パターンを大量に浴びる

人気の動画、かっこいいポスター、売れている商品をたくさん見る。「なぜこれがいいのか」を考えると、選ぶ力が育ちます。

パターンを知らない逆張りは外しやすい
04

曖昧でも決めて進める

完璧な正解を待ちすぎない。「まずこれでやってみよう」で進める。やってから直すほうが、ずっと速く成長できます。

ドライブ力は全職種で強い価値になる
05

感じの良い人になる

あいさつする、返事が早い、感じよく話す。結局、最後は「この人とまた組みたい」がすごく強いです。

人間力は甘い話ではなく、競争力
AI時代の勝ち筋 =
「人にしかできない価値」 × 「AIで増幅する力」

すごく簡単にまとめると、
AIは「手を速くする道具」、人は「何をやるかを決める存在」です。
作るだけの人ではなく、選ぶ・意味をつける・人を動かす人になるほど、これから強くなります。
