AIエージェントってなに？

チャットAI ＝「お店の店員さん」に相談するイメージ。AIエージェント ＝「家に来てくれるお手伝いさん」。自分のPCの中身を見て、ファイル整理・予約・メールまで実際にやってくれる

Coworkをインストール＆初体験

全員でダウンロード → Proプラン課金（月約3,000円）→ デスクトップに練習フォルダ作成 → プロンプトを貼るだけでフォルダやファイルを自動生成してくれるのを体験！

claude.md ＝ 玄関のルールブック

自分のプロフィールやルールを書いておくと、Claudeが毎回読んでくれる。使うほど「自分専用AI」に育っていく仕組み

Claude in Chrome でブラウザ操作

Chrome上のページをClaudeに操作させる拡張機能。求人サイト60件をリスト化、映画チケット予約、フライトキャンセルまで実演！

スマホ連携（Dispatch）

外出先からスマホでPCのClaudeに指示を出せる。ちょっとした確認はスマホのチャット版で、重要な作業は帰宅後にPC版で

トークンの仕組みを理解

5時間ごとにリセットされる枠 ＋ 週間上限の2段構え。チャットをこまめに新規作成すると節約できる。モデルは3種から選択可

Claude Code にも触れてみた

Coworkと同じAIエンジンの「開発者向け入口」。ターミナルから動かすぶん自由度が高く、Googleカレンダー連携や「毎朝タスク整理してくれる秘書」の設定もできる。まずはCoworkから慣れて、興味が出たらCode側も！

💬 みんなの「これどうなの？」

Q CoworkとClaude Codeって何が違うの？
A 中身は同じAIエンジン！Coworkは「AIに仕事を丸投げできるモード」で選択肢UIがあって初心者にやさしい。Claude Codeは「AIと一緒にコードを書くモード」で開発者向け。上位・下位ではなく、入口が違うだけ！
Q フォルダを指定する意味って？
A Claudeの作業場所（＝「お家」）を決めること。範囲を絞るとトークン節約にもなる。必要なら「PC全体見て」って指示すれば範囲外も見てくれるよ
Q ひとつのチャットでずっと話し続けていいの？
A 長く続けると精度が下がる＆トークンも食う。話題が変わったら新しいチャットを開くのがコツ。claude.mdに書いてあることは引き継がれるから安心
Q Chrome拡張でログイン済みのサイトも使える？
A Chromeでログインしてるサイト（Googleドライブ等）はそのままアクセスできる。ただしInstagram等のSNSはブロックされることがあるので向き不向きあり
Q 精度を上げるにはどうしたらいい？
A 「育成ゲーム」だと思って！日記、ブログ、MBTI、ストレングスファインダー…自分の情報を入れるほど「自分専用AI」に進化していくよ
