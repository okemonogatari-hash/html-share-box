# HTML図解 公開用ボックス

このフォルダは、**Notion に埋め込める URL 用の箱**として使う前提の公開用フォルダです。

## 役割

- `diagrams/`  
  図解そのものを置く場所

- `index.html`  
  公開図解の入口ページ

## 今入っているもの

- `diagrams/notebooklm-genspark-workflow.html`
- `index.html`

## 使い方

1. 元HTMLはいつも通り `060_Tools/制作物/HTML制作物/` で作る
2. 公開したいものだけを `public_box/diagrams/` にコピーする
3. 必要に応じて `index.html` に追加する
4. この `public_box` を GitHub Pages などに載せる

## Notion 埋め込みの考え方

Notion に埋め込みたいのは、この中の **公開URL** です。

たとえば公開後は、こんな形になります。

- 入口ページ: `https://.../index.html`
- 図解単体: `https://.../diagrams/notebooklm-genspark-workflow.html`

Notion に埋め込む時は、図解単体URLを使うのが分かりやすいです。
